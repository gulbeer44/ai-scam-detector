# 🚨 AI Scam Detector

An intelligent web application that detects scam messages using Machine Learning (BERT) combined with rule-based analysis.

---

## 🔹 Features

- ✅ Detects Scam vs Safe messages
- 📊 Confidence score & risk level
- 🏷 Scam type classification (Phishing, Investment, etc.)
- 📈 Analytics dashboard
- 🔐 Secure authentication (JWT based)
- 📄 Downloadable reports

---

## 🛠 Tech Stack

### Frontend
- React.js
- Material UI

### Backend
- Node.js
- Express.js

### AI Service
- Python
- FastAPI
- BERT (Transformers)
